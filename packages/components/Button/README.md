# Button

<!-- STORY -->

## Installation

```shell
$ yarn add @roo-ui/button
```

## Example

```jsx
import Button from '@roo-ui/button';

export default <Button>Hello world</Button>;
```

## Properties

| Name       | Description    | Type   | Default | Required? |
|:-----------|:---------------|:-------|:--------|:----------|
| `onClick`  | click callback | `func` |         |           |
| `children` | button content | `node` |         | ✔︎         |
